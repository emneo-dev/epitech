# 108trigo
An Epitech Math Project

## Usage

```
USAGE
    ./108trigo fun a0 a1 a2

DESCRIPTION
    fun     function to be applied, among at least "EXP", "COS", "SIN", "COSH"
            and "SINH"
    ai      coeficients of the matrix
```
